# 프로젝트 개요
- 블라인드 기반 기업 리뷰/커뮤니티 서비스
## 개발 기간
- 2025.08 ~ 2025.09
## 목적
- 사용하지 않았던 기술 학습 (Kafka, ReplicationDB)
## 핵심 기능
- Kafka 기반 메시지 서비스
- Replication DB: Master/Slave 읽기/쓰기 분리
- RefreshToken/AccessToken 기반 회원 관리 보안
